# 🛒 E-commerce SQL Analysis Project

## 📌 Overview
This project analyzes an e-commerce dataset to understand customer behavior, product performance, and sales trends using SQL.

## 🛠 Tools Used
- MySQL
- SQL (Joins, Aggregations, Window Functions)

## 📊 Key Business Questions
- Who are the top customers by revenue?
- What are the best-selling products?
- How do sales change over time?
- What is the average order value?

## 🔍 Key Insights
- A small percentage of customers generate most revenue
- Certain products consistently outperform others
- Sales patterns can be tracked across time

## 📁 Files in this Repository
- ecommerce_analysis.sql → SQL queries used for analysis

## 🚀 Skills Demonstrated
- Data cleaning and analysis using SQL
- Writing complex JOIN queries
- Using aggregations and window functions
- Extracting business insights from data
